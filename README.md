# RuneBarMoP

Simple adjustable Rune bar for Death Knights in MoP Classic.

## Information

- **Version:** 1.1
- **CurseForge:** https://www.curseforge.com/wow/addons/runebarmop

## Installation

1. Download the latest release
2. Extract to `World of Warcraft\Interface\AddOns\`
3. Restart WoW
